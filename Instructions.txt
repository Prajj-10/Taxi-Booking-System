Hello ! 
To run this project please install the following external libraries:

- Pillow (pip install Pillow)
- TkCalendar (pip install tkCalendar)
- pymsql (pip install pymsql)

Images for the GUI are also in the folder called Images

The database is located in the MySQL folder
By running the queries the database will be up and running in the local host.
